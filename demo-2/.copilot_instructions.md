# GitHub Copilot Instructions for MapsterCodeMaze

## Language Requirements

- **Always use English** for all comments, documentation, and code explanations
- Do not use Vietnamese, Chinese, or any other languages in comments
- All method documentation and inline comments must be in English

## Code Style Guidelines

- Follow C# 7.3 conventions (this project targets .NET Framework 4.8)
- Use clear, descriptive variable and method names in English
- Maintain consistency with the existing codebase style
- Keep comments concise but informative

## When Generating Code

- Generate all comments in English
- Use proper code formatting with appropriate indentation
- Follow the existing patterns established in this project (Mapster mappings, DTO patterns, etc.)
- Include meaningful comments for complex logic
